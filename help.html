    Common Questions

    What is Andro?

    Andro is a website which categorizes different websites based on their traffic, given grade, industry, and more. You can filter websites by typing in the searchbar to make it easier when an emergency occurs.

    Which browsers do you support?

    At the moment, we aim to support all major web browsers. Any issue in the browsers listed below should be reported as a bug .

       -  Microsoft Edge 14+
       -  Firefox (Current - 1) and Current versions
       -  Chrome (Current - 1) and Current versions
       -  Opera (Current - 1) and Current versions

    (Current - 1) and Current means that we support the current stable version of the browser and the version that precedes it.

    What are the compatible devices?

    At the moment, we aim to support all major devices. Any issue in the device-types listed below should be reported as a bug .

       -  High Resolution Desktops
       -  Regular Desktops and Laptops
       -  Low Resolution Tablets and Smartphones
       -  Majority of Smartphones
       -  Modern Tablets and iPads

    Is this software open-source?

    Andro is and will be open-source forever. Open source licensing encourages innovation through collaboration. Without it, many of the technologies we take for granted today would never have developed, or would been locked away behind patent law.

    What license do you use?

    Andro is released under the MIT License. You can learn more About it here.

    What's next?

    We plan on adding more features along the way, as well as new websites and continue evolving Andro through with the help of your contributions. To learn more About how you can contribute, visit this page.

    How can I contribute?

    The main purpose of making Andro open-source is to continue evolving it, making it faster and easier to use. You can contribute by closing an issue, fixing a bug, submiting new feature ideas, improving the codebase, adding new website entries, and so on. To learn more, please Contact Us, or visit our Contributing page for more in-depth information.
