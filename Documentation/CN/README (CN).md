# Andro

Andro 让您通过每周一次的手选定类别和充满新的有趣和有用的网站的趋势来找到新网站， 你可能不知道的那种情况。

## 国际化

我们目前支持10种语言，包括英语。 我们是否缺少您的语言？ [通过 Crowdin](https://crowdin.com/project/andro-dashboard) 贡献翻译！
